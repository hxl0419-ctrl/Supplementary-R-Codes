### This is the code file for the paper "Dynamic Count Time Series Modeling and Anomaly Detection for Online Automotive Quality Complaints".

### Time:  2025.09.20

Code File and Description：

File name: Brake_INAR_fit
Description: This R script performs the following analyses: it applies the Ljung-Box test, the Augmented Dickey-Fuller (ADF) test, fits and compares eight different INAR(1) models with various innovation distributions to the dataset of monthly brake noise complaint counts.

File name: transmission_INAR_fit
Description: This R script performs the following analyses: it applies the Ljung-Box test, the Augmented Dickey-Fuller (ADF) test, fits and compares eight different INAR(1) models with various innovation distributions to the dataset of monthly transmission noise complaint counts.
